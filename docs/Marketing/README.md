# BitWealth — Brand & Marketing Kit

**Version 1.0**

This kit contains everything needed to represent the BitWealth brand accurately and
consistently across client-facing documentation, the website, and marketing materials.

## Logo asset library

The kit ships with the most-used logo variants in `assets/logos/`. The **complete**
source library — including the full favicon set, Apple touch icons, and Android/PWA
icons at every size — lives in the repository `/logos` folder:

- `bitwealth_logo_transparent*.png / .svg` — full lockup for light backgrounds
- `bitwealth_logo_white*.png / .svg` — reversed lockup for dark/navy backgrounds
- `Website & UI Icons/` — icon-only marks (64–256 px)
- `Apple Touch Icon/` — 180×180 app icons
- `Android & PWA Icons/` — 192, 384, 512 px icons
- `Favicon/` — favicon set

## Notes for partners

- Company statistics that appear on the current website (e.g. founding year, assets
  under management, client counts, uptime) are **placeholders** and should not be
  reproduced in client-facing material until confirmed by BitWealth.
- Always include the standard risk disclaimer on any performance-related material
  (see [02_Messaging_and_Voice.md](02_Messaging_and_Voice.md)).
- A separate regulatory/compliance section is not yet part of this kit — request it
  from BitWealth before publishing any regulated financial-services claims.
